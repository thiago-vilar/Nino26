# FASE3-NINO — reconstrução científica

Série de 11 notebooks baseada em ONI local compatível para ocorrência, P90 somente para intensidade, quatro fases retrospectivas, estatística por evento, redução de redundância, PCA por fase, mapas OISST, Kelvin/vento, Hovmöller e sensibilidade.

Todas as tabelas completas são salvas em `data/processed/numeric-tables/fase3_nino_cientifica/` e todas as figuras em `data/processed/figures/fase3_nino_cientifica/`, com 700 ppi. Os notebooks mostram somente as primeiras 12 linhas das tabelas.
